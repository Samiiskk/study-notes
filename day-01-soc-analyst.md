## What is a SOC?
A Security Operations Center is a team that monitors,
detects, and responds to security threats 24/7.
They watch over an organization's systems, networks,
and data in real time.

## What does a Jr. SOC Analyst actually do?
- Monitor security alerts from SIEM tools
- Triage incoming alerts — decide what is real vs false positive
- Escalate serious incidents to senior analysts
- Document everything — every alert, every action taken
- Work in shifts as SOC runs 24/7

## Tiers in a SOC team
- **Tier 1** — Alert triage, monitoring (Jr. Analyst level)
- **Tier 2** — Deeper investigation, incident response
- **Tier 3** — Threat hunting, advanced analysis

## Key tools a SOC Analyst uses
- **SIEM** — collects and analyzes logs (e.g. Splunk, IBM QRadar)
- **IDS/IPS** — detects/prevents intrusions
- **Firewall** — controls network traffic
- **EDR** — monitors endpoint devices
